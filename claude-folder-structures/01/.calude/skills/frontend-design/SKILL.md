# Frontend Design Standards

This skill defines the visual language and design principles for all frontend interfaces. Use these settings to ensure consistency across the application.

## Core Visual System

### 🎨 Colors
- **Primary:** `#FF6B35`
- **Background:** `#0A0A0F`
- **Surface:** `rgba(255, 255, 255, 0.04)`

### 🖋️ Typography
- **Primary Font:** `Inter`
- **Tracking:** `-0.02em`
- **Hero Headers:** `48-64px`
- **Body Text:** `15-16px`

### 📏 Spacing & Geometry
- **Section Spacing:** `64px`
- **Card Padding:** `24px`
- **Border Radius:** `16px`

### 🌑 Dark Mode Principles
- **No Flat Black:** Never use `#000000` for backgrounds.
- **Depth:** Create visual hierarchy through subtle gradients, glows, and thin borders.
- **Micro-interactions:** Add subtle glassmorphism and light effects for more tactile feel.
